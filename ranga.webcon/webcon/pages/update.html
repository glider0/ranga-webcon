使用 NSWA Online 推送更新系统

如果你没有关闭 Web 控制台的 NSWA Online 推送功能，新版本发布后，在你打开 Web 控制台时会向你推送。弹出更新通知。

使用命令行工具更新系统到最新版本

使用命令行工具可以自动将 NSWA Ranga 系统更新到最新版本，这在需要安装多个 OTA 更新的时候非常有用。大部分情况下，使用此命令可以将 NSWA Ranga 系统自动安装从当前版本直到最新版本所有的 OTA 更新包。 
ranga-cli auth -p
ranga-cli swdeploy sync
如果需要使用命令行工具安装一个固件文件，你可以 
ranga-cli auth -p
ranga-cli swdeploy flash /path/to/firmware.bin
有关命令行工具进行系统更新的更多用法，请运行 ranga-cli swdeploy -h 获得帮助信息。

使用 Web 控制台上传固件

通过 Web 控制台上传系统固件更新是具有实验性的，且你的浏览器必须足够新以支持目前仅仅是 Web 工作草案的 File API。"下次完整包更新时抹掉当前系统配置" 会在你（主动或被动）重启系统之前一直有效；"礼貌地要求安装完毕后执行重启" 并非一个强制性的操作，而是向更新程序提出请求，可以被更新程序忽略，同时，更新程序也可能自行决定是否应该重启，而不管你是否请求，但是，对于大部分无需重启的更新，使用此功能将会在更新完毕后重启。
