The scheduled task service will not start automatically by default. You can go to Miscellaneous and tweaks to enable the function of the automatic start scheduled task service.
The modification of the scheduled task settings will not take effect until the NSWA Ranga system is restarted or the CRON service is restarted, so after modifying these settings, you can click the "Restart CRON Service" button to take effect immediately. This operation takes a little while to complete, you don't have to modify each configuration to execute, you can complete multiple configurations of the scheduled task and then execute it once.

When do you want to wake up this scheduled task?

Use '*' at any interval, use 'number/*' every time, use 'number' for specific time, use 'number-number' for multiple consecutive specific times, use multiple numbers, numbers, numbers, etc. .'. Minutes 0 to 59, hours 0 to 23, days 1 to 31, months 1 to 12, and weeks 0 to 7.

Example:
Execute every 1 minute: * * * * *
Execute every 1 hour: 0 */1 * * *
Execute every 3 hours: 0 */ 3 * * *
Executed at 6am every day: 0 6 */1 * *
Execute once every 4th and 18th minutes: 4,18 * * * *
At 4am Execute once in the 5th and 35th minutes of 11:5,35 4-11 * * *
Execute once every 11pm on Saturday: 0 23 * * 6
Every Saturday, Sunday 18 o'clock Execute every 30 minutes between 23:0,30 18-23 * * 6,0
